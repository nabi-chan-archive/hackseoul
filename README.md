# 🚀 Requirements

- `bun@1.1.22`
- `node@20.16.0`
- `.env.local` (if you required environments for testing, just dm @nabi-chan in discord)
  - `supabase API key`
  - `openAPI key`

# 🔨 How to run

```bash
# install dependencies
bun install

# running dev server
bun dev
```
